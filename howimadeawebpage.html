HTML

the languages used to code the webpage are obviously html and css cuz i think those are the standard.
when setting up the basic formula of each webpage i used a container [div] to make a small box in the middle. this is where you are reading rn btw. I used an exact pixel width instead of a percentage width so it doesnt squish on mobile [speaking of which its pretty much the width of a phone screen so its compatible!!]

but a plain color in the background is BORING, so i used some background css elements to make an cool galaxy background

[EB]: you can also use font face and color elements in html to have segments be different colors.


FONT

with a basic font-family element, you can give the page or certain text elements different fonts! however the only ones from the start are default microsoft fonts :[

but you can use downloaded fonts like Determination Mono [this one!] or anything really using an @font-face thingy in the css document. yu give it a nickname to refer to when using it and the installer directory and it works! :3


IMAGES [href/src, flex]

no webpage is complete without a funky image ;] so i use img elements to reference files on my computer [.gif or .jpg] but the blinkies on the main page are not in the repository, so they link to an image link, which can be copied from rightclicking most images online i think.

linking is also important because hyperlinks use href while images use src btw. the repository jump links to the heading that has the id 'github', which is mentioned with '#github' as the link

if i just put an img element followed by a p[paragraph] they would stack. there are other ways to fix this but i use the flex property. i put both the image and text elements into a container with the display:flex; property, and they get squished in one line. width and height paramaters and stuff help this look not weird btw
